# 📚 Ciclo Formativo DAW

Repositorio donde organizo y almaceno el contenido, apuntes, ejercicios y proyectos desarrollados durante el **Ciclo Formativo de Desarrollo de Aplicaciones Web (DAW)**.

Este repositorio está estructurado por módulos, cada uno con sus respectivos ejercicios prácticos, ejemplos, teoría y proyectos realizados durante el curso.

## 🎯 Objetivos

- 📖 Documentar mi aprendizaje  
- 💻 Centralizar prácticas y proyectos  
- 🚀 Mostrar mi evolución como desarrollador  
- 🗂️ Mantener una estructura clara y organizada

## 📘 Módulos

### 💻 Programación

Módulo enfocado en el aprendizaje de **Java** y los fundamentos de la programación.

**Contenidos principales:**

- Fundamentos de programación  
- Programación Orientada a Objetos (POO)  
- Estructuras de control  
- Arrays y colecciones  
- Manejo de excepciones  
- Ficheros  
- Buenas prácticas de programación  

**Tecnología utilizada:**

- Java  

---

### 🗄️ Base de Datos

Módulo centrado en el diseño y gestión de bases de datos relacionales.

**Tecnologías utilizadas:**

- MySQL  
- Oracle Database  

**Contenidos principales:**

- Modelo Entidad-Relación  
- Normalización  
- SQL (DDL, DML, DCL)  
- Consultas avanzadas  
- Procedimientos almacenados  
- Triggers  
- Diseño y administración de bases de datos  

---

### 🛠️ Entornos de Desarrollo

**Contenidos principales:**

### 🏷️ Lenguaje de Marcas
